# Bank Customer Segmentation Analysis Dashboard

## Project Overview
This project analyzes customer demographics, income levels, tenure, loyalty segments, and growth trends using Microsoft Excel. The objective is to identify customer groups, understand customer behavior, and generate insights that can support targeted marketing and customer retention strategies.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Dataset Information
The dataset contains customer-related information, including:

- Customer ID
- Age
- Gender
- Income
- Tenure
- Occupation
- Region
- Customer Type
- Loyalty Segment

## Dashboard Metrics
- Total Customers
- Average Age
- Average Income
- Average Tenure

## Analysis Performed

### Customer Generation Analysis
Analyzed customer distribution across generations:
- Boomers
- Gen X
- Gen Z
- Millennials

### Customer Loyalty Segmentation
Classified customers into:
- Loyal
- Legacy
- Emerging
- New

### Income Bracket Analysis
Segmented customers based on income levels:
- Low Income
- Medium Income
- High Income

### Occupation Analysis
Studied customer distribution across professions:
- Banker
- Doctor
- Engineer
- Retired
- Student
- Teacher
- Trader

### Customer Growth Analysis
Analyzed year-wise customer growth trends from 2015 to 2025.

## Dashboard Features
- Interactive KPI Cards
- Dynamic Pivot Charts
- Region-wise Filtering
- Customer Type Filtering
- Loyalty Segmentation Analysis
- Income Distribution Analysis
- Customer Growth Timeline

## Key Insights
- The dataset contains 500 customers with an average age of 46 years.
- Loyal customers account for approximately 40% of the customer base.
- Low-income customers form the largest customer segment.
- Customer growth peaked in 2017 and remained stable in later years.
- Customer demographics and occupations show diverse representation across customer groups.
